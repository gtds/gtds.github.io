Introduction

GTDS is a private server for the game called 'Growtopia' Below are the tutorial on how to join the server.

Windows
  • Click window key or click the windows icon on bottom left
  • Find notepad
  • Right click notepad and run as administrator
  • Go to File > Open or CTRL + O in notepad
  • Go to C:\Windows\System32\drivers\etc or paste the following path at the top. %systemdrive%\Windows\System32\drivers\etc
  • Click file type and select all files
  • Choose the file named "hosts"
  • Now add the following to the text. 38.107.214.220 growtopia1.com
    38.107.214.220 growtopia2.com     to the bottom.
  • Save the file (CTRL + S)
